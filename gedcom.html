Importar / Exportar GEDCOM

A importação e exportação de GEDCOM é feita nas definições da árvore.

Definições da Árvore

O que é o GEDCOM?

Tudo o que precisa de saber sobre este padrão de genealogia

O que é?

GEDCOM (Genealogical Data Communication) é um formato de ficheiro de texto criado para guardar e partilhar dados genealógicos — como pessoas, datas, locais e relacionamentos familiares.

Para que serve?

Serve como "língua comum" entre diferentes programas de genealogia. Ao exportar o seu arquivo em GEDCOM, pode abri-lo em qualquer outra aplicação compatível sem perder informação.

Quem usa?

Genealogistas de todo o mundo, desde utilizadores domésticos a arquivos históricos e centros de investigação. Plataformas como Ancestry, FamilySearch, Gramps e MyHeritage suportam GEDCOM.

Quando surgiu?

Foi criado em 1984 pela Igreja de Jesus Cristo dos Santos dos Últimos Dias (FamilySearch) para digitalizar e preservar registos genealógicos. Ao longo dos anos evoluiu com novas versões, sendo a mais recente a GEDCOM 5.5.5 e GEDCOM 7.

Como funciona, em termos simples?

1

Um ficheiro .ged é um ficheiro de texto simples que qualquer editor de texto consegue abrir. Dentro, cada linha começa com um número (o nível), seguido de uma etiqueta e o valor — por exemplo: 1 NAME João /Silva/.

2

Cada pessoa (INDI) e cada família (FAM) tem um identificador único. As relações entre pais e filhos são estabelecidas por referências cruzadas entre esses identificadores.

3

Ao importar um ficheiro GEDCOM nesta aplicação, todos os registos são lidos e convertidos para a base de dados local. Ao exportar, acontece o inverso — os seus dados são convertidos para o formato texto do GEDCOM.

4

Pode usar o ficheiro exportado como cópia de segurança, para partilhar a sua árvore com familiares, ou para migrar os dados para outra ferramenta de genealogia.

Entidade responsável pela norma GEDCOM

O padrão é atualmente mantido pela FamilySearch, uma organização sem fins lucrativos. A especificação completa e as versões mais recentes estão disponíveis no repositório oficial.
